# Искра: Ядро

**Версия:** 1.0.0
**Тип:** Модуль
**Code:** `iskra_core`
**Совместимость:** OpenCart 4.1+

---

## Описание

Iskra Core — фундаментальное расширение экосистемы Искра. Предоставляет базовую инфраструктуру для всех остальных расширений Искра: логирование, шину событий, очередь задач, расширенные настройки, миграции базы данных, проверку целостности и автоматические бэкапы.

---

## Возможности

- **Система логирования** — Структурированные логи с 5 уровнями (debug, info, warning, error, critical), категориями, отслеживанием пользователей и полной поддержкой кириллицы.
- **Глобальный переключатель логирования** — Включение или отключение записи логов без удаления расширения.
- **Авто-ротация логов** — Очистка с учётом хранилища, 3 стратегии:
  - `rows` — лимит по количеству строк
  - `table_size` — лимит по размеру таблицы (МБ)
  - `disk_space` — лимит по свободному месту на диске (МБ)
- **Event Bus** — Синхронная и асинхронная коммуникация между расширениями через события.
- **Очередь задач** — Фоновая обработка задач с приоритетом, попытками и планированием.
- **Миграции БД** — Версионирование изменений схемы с отслеживанием, контрольными суммами и откатом.
- **Проверка целостности** — Предварительная и пост-установочная проверка всех таблиц Искра.
- **Система бэкапов** — Авто-бэкап перед установкой/удалением, ручное создание бэкапов, восстановление и управление сроком хранения.
- **Поддержка кириллицы** — Полная кодировка UTF-8 MB4 для русского, украинского, казахского, белорусского и румынского языков.

## Установка

### Через админ-панель OpenCart (рекомендуется)

1. **Система → Расширения → Установщик**
2. Загрузить `iskra_core-1.0.0.ocmod.zip`
3. **Система → Расширения → Расширения → Модули**
4. Найти **Искра: Ядро** → нажать **Установить**
5. Нажать **Изменить** для настройки

### Ручная установка

1. Скопировать `extension/iskra_core/` в корень OpenCart
2. Перейти в **Система → Расширения → Расширения → Модули**
3. Найти **Искра: Ядро** → **Установить**

---

## Админ-интерфейс

После установки:
**Расширения → Расширения → Модули → Искра: Ядро → Изменить**

### Вкладки

- **Настройки** — Переключатель логирования, уровни логов, срок хранения, стратегии ротации, Event Bus, Очередь, хранение бэкапов.
- **Логи** — Просмотрщик логов с фильтрами, сортировкой, пагинацией, массовым удалением и ручной ротацией.
- **Бэкапы** — Создание, восстановление, скачивание и удаление бэкапов базы данных.
- **Миграции** — Просмотр статуса миграций, запуск ожидающих, откат последней миграции.
- **Целостность** — Проверка существования таблиц, количества строк, размеров и кодировки.

| `oc_iskra_log` | Логи приложения |

## Удаление

**Система → Расширения → Расширения → Модули → Искра: Ядро → Удалить**

⚠️ **Внимание:** При удалении будут удалены все таблицы Искра и настройки. Перед удалением автоматически создаётся бэкап в `storage/backup/iskra_core/`.
